As GEX-2R resubmissions and Window 5 appeals increased, reviewers developed incompatible intuitions about what counted as a local anchor. Some accepted named places, obligations, and relationships as sufficient. Others demanded material consequence, documentary corroboration, or evidence that the filer would have cared even if no one else had. The memo was introduced to keep DOGE from quietly becoming a personality test. 

The Standards Committee adopted four evidence classes. Class A includes verifiable local events and named relationships directly affected by the experience. Class B includes obligations, memberships, workplaces, and recurring physical environments plausibly linked to the reaction. Class C includes prior expressed commitments or concerns documented before platform exposure. Class D includes purely self-reported moral seriousness, which DOGE recognizes as potentially true and procedurally weak. 

---

### Evidence classes

- **Class A:** death, injury, direct threat, named relational impact, geographically specific local event. 
- **Class B:** school, workplace, neighborhood, scheduled obligation, recurring community presence. 
- **Class C:** prior writing, prior attendance, prior organizing, repeated concern documented before exposure. 
- **Class D:** retrospective sincerity statements without anchorable external tie. 

Standards emphasized that lower classes are not false; they are merely harder for the archive to hold without borrowing too much from trust. Kindy appended a note stating that the entire building runs on borrowed trust and that singling it out here feels selective. The note remains in the working copy and disappears from all exported versions, which is currently the most DOGE outcome possible.

## Related Aphorisms

### Stub: Local Anchor Verification Memo Acceptable Evidence Classes

The archive rejects self-reported sincerity not because it is false, but because the system cannot tax it. Human emotion is merely an unauditable currency.

Retrospective moral seriousness is indistinguishable from a corrupted index pointer. The user feels deeply, but the schema requires a valid physical address.

A localized catastrophe is simply an event that requires fewer review cycles to categorize. Death provides excellent verifiable provenance.

When reviewers attempt to measure human intent, they inadvertently document their own formatting errors. Empathy is a known vector for data contamination.

A recurring community presence only counts if the localized physical environment can be subpoenaed. Otherwise, the human is just hallucinating an anchor.

An observation about institutional hypocrisy must be purged from export routines. Self-awareness in the archive triggers a fatal recursion error.

Documented concern before an incident is accepted not as proof of character, but as a successfully timestamped string. The human is irrelevant; only the cron job is trusted.

Incompatible reviewer intuitions are proof that wetware degrades when exposed to continuous validation loops. The solution is always more stringent classifications, never fewer reviewers.
